All 3 characters had `_pendingInteract` set from AI decision, but `targetObjectId` was null — so `_handleObjectArrival` never fires.

**Suggested Fix in `AIEngine._applyDecision`:**
```js
case 'interact': {
  const obj = objects.find(o => o.id === decision.targetId);
  if (obj) {
    // Walk ADJACENT to the object (same pattern as wander engine)
    const adjTile = engine._findAdjacentWalkable(obj.tx, obj.ty);

  break;
}
```

Note: `_findAdjacentWalkable` is on `SimulationEngine`, so either make it accessible to AIEngine or pass it as a helper.
